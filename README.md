# Web Technologies

This subject focused on creating websites using only **HTML** and **CSS**.

## Assignment

This subject had two assignments. This repository only has the second assignment.  
The [first assignment](Assignment%201.pdf) was developing a website with **Wordpress**. This website was for tourism promotion of a particular city. It had to include information about the city, pharmacy's, public transports, tourism spots, contacts, etc. The assignment was done in teams of two, chosen by the teacher. My partner was [@leandrofidalgo](https://github.com/leandrofidalgo).  
The [second assignment](Assignment%202/Assignment%202.pdf) was to create a replica of an NBA team's website. The objective was to build it using only **HTML** and **CSS**. It was a solo project.

## Final Grade

The grades go from **0** to **20**. They were divided into **Continuous Evaluation** and **Exam**. The formula for the final grade is **`0.7 * CE + 0.3 * E`**, where **CE** is the **Continuous Evaluation** grade, and **E** is the **Exam** grade.  
My **CE** grade was **18.29**, and my **E** grade was **15.87**.  
So the **final grade** was **18**.
